throttle module Documentation

Release Notes

Record

throttleRecord.html
The throttle record is used for limiting how often a value is sent to a PV. Values can be continuously be written to it, but only after the given delay will the latest value be sent.

EPICS Database

throttle.db
This database is basically a straight implementation of the throttle record itself. It creates a record of the name "$(P)$(THR)".

MEDM display files

throttle.adl
This is the medm ADL file for an throttle record. All the fields are exposed on this screen.

How to build and use

  • Edit configure/RELEASE to specify the paths to EPICS base. There are no dependencies on any other modules.
  • Run Gnu Make to build.
